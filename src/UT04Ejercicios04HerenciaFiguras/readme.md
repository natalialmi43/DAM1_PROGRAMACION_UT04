# UT4 - Ejercicios 4 - Herencia sencilla - no evaluable

Vamos a crear una aplicación para el cálculo de áreas de figuras.
Todas las figuras van a tener un color.
En cada figura concreta se deberá calcular el área y el perímetro.
Vamos a disponer de varias figuras: cuadrado, triángulo y círculo.
Se deberá crear el árbol de herencia adecuado para este problema.
