# UT4 - Ejercicios 5 - Herencia - vehiculos - no evaluable
Vamos a crear una aplicación para la clasificación de vehículos.
Todas los vehículos van a tener un color, modelo, precio y marca.
Podemos disponer de vehículos con y sin motor:
Vehículos con motor van a disponer de cilindrada y combustible.
Vehículos sin motor se debe almacenar el material.
Los dos tipos de vehículos van a disponer de constructores con todos los atributos.
Como vehículos concretos hay:
Bicicleta que tendrá un número de marchas.
Coche con un número de puertas.
Moto que almacena el tipo.
Patinete con un número de ruedas.
Al igual que con los vehículos, todos estos dispondrán de constructores con todos los atributos.
Se deberá crear el árbol de herencia adecuado para este problema.
Piensa si debemos tener llamadas entre constructores.
